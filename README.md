# Dakera AI Homebrew Tap

```bash
brew tap dakera-ai/tap
brew install dk
```

## Available Formulae

| Formula | Description | Managed by |
|---------|-------------|------------|
| `dk` | Dakera CLI — manage agent memories, sessions, and vector search from the terminal | cargo-dist (auto-updated on release) |
| `dakera-mcp` | Dakera MCP server — pending binary distribution (DAK-5409/DAK-5410) | Manual |

## Platforms

`dk` supports:
- macOS (Apple Silicon + Intel)
- Linux x86_64
